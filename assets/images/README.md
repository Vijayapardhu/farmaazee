## Image Guidelines

### Quality Requirements
- Use high-resolution images (minimum 72 DPI for web)
- Optimize for web (compress without losing quality)
- Use JPG for photographs, PNG for graphics with transparency
- Keep file sizes under 500KB for optimal loading

### Content Requirements
- All images should be agriculture/farming related
- Show modern, technology-enabled farming practices
- Include diverse representation of farmers
- Use natural, earthy color schemes
- Ensure images are professional and trustworthy

### Technical Specifications
- Use RGB color space
- Save with appropriate compression
- Include alt text descriptions
- Test loading performance
- Ensure responsive design compatibility

## Image Sources

You can source images from:
1. **Stock Photo Websites**: Unsplash, Pexels, Pixabay (free)
2. **Professional Photography**: Hire photographers for custom images
3. **AI Generated**: Use AI tools like DALL-E, Midjourney for custom visuals
4. **Partner Organizations**: Collaborate with agricultural organizations
